# Stock Analysis AI Bot (Discord版)

## プロジェクト概要
ユーザーが入力した証券コードに基づき、個別銘柄の多角的な分析レポート（ファンダメンタル・テクニカル・需給・企業概要）を自動生成し、Discord上で提供するAIチャットボットです。

## 目的
投資判断に必要な情報をワンストップで提供し、分析にかかる時間を短縮する。

---

## 機能要件

### 1. 入力インターフェース
- **プラットフォーム:** Discord
- **コマンド:** `/analyze <証券コード>`


### 2. 出力レポート構成

#### A. 企業概要
- **事業内容:** ビジネスモデルの要約
- **特徴・強み:** 経済の堀 (Moat)、技術的優位性
- **事業リスク:** 外部環境、規制、地政学リスク
- **競合他社:** 比較対象企業（3社程度）

#### B. ファンダメンタルズ分析
- **財務推移 (過去5年):** 売上高、営業利益、経常利益、純利益、EPS
- **株価指標 (過去5年):** PER、ROEの推移グラフ
- **比較分析:** TOPIXおよび同業他社との比較
- **AI考察:** 収益性のトレンドと割安感の評価

#### C. テクニカル分析
- **チャート:** 週足・日足ローソク足チャート (画像生成)
- **指標:**
    - 移動平均線 (短期・中期・長期)
    - 出来高
    - RSI, ストキャスティクス (%K, %D, Slow%D)
- **AI考察:** トレンド判定、過熱感、エントリーポイントの示唆

#### D. 需給分析
- **信用残:** 買い残・売り残・信用倍率の推移グラフ
- **機関動向:** 機関投資家の空売り残高
- **その他:** 大量保有報告書、自社株買い情報
- **需給スコアリング:** S/A/B/C の4段階評価

---

## 技術スタック

### データソース
- **株価・財務・信用残:** 株・プラス (Kabu+)
- **適時開示・大量保有:** EDINET API
- **機関空売り:** 空売り集計サイト (Scraping)
- **企業情報:** Google Custom Search API

### 開発環境
- **言語:** Python 3.x
- **Bot Framework:** discord.py
- **データ処理:** pandas, numpy
- **グラフ描画:** mplfinance, matplotlib
- **AIモデル:** OpenAI API (GPT-4o) or Google Gemini API
